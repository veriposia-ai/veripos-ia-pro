# VeriPOS IA Pro

Un sistema POS avanzado para restaurantes, con IA, gestión de stock, ventas y más. Desarrollado en vanilla JS/HTML/CSS.

## Instalación
1. Clona el repo: `git clone https://github.com/tu-usuario/veripos-ia-pro.git`
2. Abre `index.html` en un navegador.

## Funcionalidades
- POS con carrito, descuentos y multi-impuesto.
- Admin para productos.
- Informes y IA simulada.
- PWA offline.
